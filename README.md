# woodswap
A spigot plugin to worldedit letting player to use a command to swap wood blocks types

## Requirements
- Spigot >= 1.16.4
- WorldEdit >= 7.2.5

## Usage
/woodswap [sourceType] [targetType]

## Permissions:
- `woodswap.swap`

## Wood types
- acacia
- birch
- dark_oak
- jungle
- oak
- spruce

## Notes
This command supports the `//undo` worldedit command
